# 欢迎来到 tnotesjs

[TNotes][] 是一个基于 [vitepress][] 搭建的开源知识库管理工具，创建于 `2024.08.28`，截止目前 `2026` 已维护一年多时间，希望分享的内容对你有所帮助！

| 知识库 | 简介 |
| --- | --- |
| [TNotes][] | TNotes 的根知识库，集成所有子库目录信息，提供知识库快速导航功能。 |

| [TNotes.c][] | C 语言笔记 |
